# Hi, I’m keepithandy

I build browser-based RPGs, reusable game-engine systems, and practical developer tools. I’m a solo developer building in public, focused on projects that are thoughtful, playable, and genuinely useful.

## Main Projects

### [DungeonDex](https://github.com/keepithandy/DungeonDex)
A browser-based roguelite/RPG project and one of my main public focuses. It’s where I explore progression, exploration, and game systems in a format that is both playable and expandable.

### [depth-engine](https://github.com/keepithandy/depth-engine)
A browser-first engine and prototype foundation for RPG-style experiences. The goal is to build reusable systems that can support future projects rather than relying on one-off implementation.

### [dev-kit](https://github.com/keepithandy/dev-kit)
A practical Python toolkit for audits, version checks, and Markdown reporting. It’s designed to make everyday development workflows a little more structured and repeatable.

## Secondary and Early-Stage Projects

- [NovaDeck](https://github.com/keepithandy/NovaDeck) — a dashboard, notes, and watchlist concept that is still early in development.
- [repair-tool-kit](https://github.com/keepithandy/repair-tool-kit) — a planned Windows maintenance and diagnostic toolkit.
- [crafting-kit](https://github.com/keepithandy/crafting-kit) — still under definition and not yet a core portfolio pillar.

## What I’m Building Toward

I’m interested in creating a body of work that connects game design, systems thinking, and developer productivity. My long-term direction is to build a reusable foundation for browser-based RPGs while also creating practical tools that help with real development work.

## Tech and Tools

I work across a mix of browser-based development, scripting, and tooling, including:
- JavaScript/TypeScript for interactive web experiences
- Python for automation, audits, and developer utilities
- Markdown and lightweight documentation workflows
- GitHub for building in public and sharing progress

## Current Focus

Right now I’m focused on sharpening the core ideas behind DungeonDex, strengthening the reusable foundations in depth-engine, and developing dev-kit into something genuinely useful for everyday work. I’m keeping the process open, iterative, and grounded in what I can build and improve over time.
